# Netflix Data Analysis

## Overview

This project analyzes Netflix dataset to explore content trends, top countries, and distribution between movies and TV shows.

## Tools Used

* Python
* Pandas
* Matplotlib

## Key Analysis

* Movies vs TV Shows distribution
* Top producing countries
* Content trends over years

## Key Insights

* Movies dominate the platform
* The United States produces the most content
* Content production increased significantly after 2015
